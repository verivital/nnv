# NNV 3.0: Major Contributions and New Features

**Full documentation:** [verivital.github.io/nnv](https://verivital.github.io/nnv/)

NNV3 introduces major new verification capabilities:

- **VolumeStar**: Verification of video and 3D volumetric data — [Theory](https://verivital.github.io/nnv/theory/imagestar-volumestar.html)
- **GNNV (GraphStar)**: Graph neural network verification (GCN, GINE) — [Theory](https://verivital.github.io/nnv/theory/gnn-reachability.html) · [Tutorial](https://verivital.github.io/nnv/examples/gnn.html)
- **ModelStar**: Weight perturbation analysis (quantization, hardware errors) — [Theory](https://verivital.github.io/nnv/theory/weight-perturbation.html) · [Tutorial](https://verivital.github.io/nnv/examples/weight-perturbation.html)
- **FairNNV**: Formal fairness certification — [Theory](https://verivital.github.io/nnv/theory/fairness.html) · [Tutorial](https://verivital.github.io/nnv/examples/fairness.html)
- **Probabilistic Verification**: Conformal prediction-based analysis — [Theory](https://verivital.github.io/nnv/theory/probabilistic.html)
- **Time-Dependent Networks**: Variable-length time series verification
- **Malware Detection Benchmark** — [Tutorial](https://verivital.github.io/nnv/examples/malware.html)

See the [NNV3 Summary](https://verivital.github.io/nnv/nnv3.html) for a complete overview, the [14-tool comparison table](https://verivital.github.io/nnv/user-guide/architectures.html#nnv-vs-other-tools), and the [Application Domains](https://verivital.github.io/nnv/application-domains.html) page.

For citations, see [How to Cite](https://verivital.github.io/nnv/publications/citing.html).
