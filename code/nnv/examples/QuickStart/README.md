# NNV Quick Start

**Full guide:** [Quick Start Documentation](https://verivital.github.io/nnv/getting-started/quickstart.html)

## Files

| File | Description |
|------|-------------|
| `test_installation.m` | Checks NNV version, Star class, layers, and NN creation |
| `simple_verification.m` | Complete verification workflow: create network → define input set → reach → verify |

## Usage

```matlab
cd examples/QuickStart
test_installation       % Verify setup
simple_verification     % First verification
```

Run `install.m` on first use. If you restart MATLAB, run `startup_nnv.m` to restore paths.
