# NNV Examples

**Detailed walkthroughs:** [Examples & Tutorials](https://verivital.github.io/nnv/examples/)

## Folder Structure

- `QuickStart/` — Installation verification and first steps
- `Tutorial/` — Beginner-friendly walkthroughs (NN, NNCS, SPIE medical imaging)
- `NN/` — Advanced neural network examples (CNN, RNN, GNN, NeuralODE, BNN, medmnist)
- `NNCS/` — Advanced control system examples (ACC, EmergencyBraking)
- `Submission/` — Paper and competition reproduction code (ARCH-COMP, VNN-COMP, CAV, etc.)
