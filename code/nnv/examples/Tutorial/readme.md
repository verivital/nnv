# NNV Tutorials

**Full documentation:** [Conference Tutorials](https://verivital.github.io/nnv/examples/conference-tutorials.html)

## Running Tutorials

**Option 1: Local installation** — See [Installation Guide](https://verivital.github.io/nnv/getting-started/installation.html), then run `startup_nnv.m`.

**Option 2: MATLAB Online** — [Open NNV folder](https://drive.mathworks.com/sharing/980b6270-7a14-451c-924c-ae07745fa279) → "Copy Folder" (may take 15 min to a couple of hours).

**Option 3: CodeOcean** — [NNV3 capsule](https://codeocean.com/capsule/6810863/tree/v1) (runs in browser).

## Tutorial Contents

- **NN/MNIST/** — Image classification robustness
- **NN/GTSRB/** — Traffic sign verification
- **NN/ACAS Xu/** — ONNX + VNNLIB safety verification
- **NN/malware/** — Cybersecurity verification
- **NNCS/ACC/** — Adaptive cruise control (requires Simulink for training)
- **NNCS/AEBS/** — Emergency braking
- **NNCS/InvertedPendulum/** — Classic control benchmark
- **SPIE/** — Medical imaging (2D classification, 3D volumetric, segmentation)
